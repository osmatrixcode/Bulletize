Troubleshooting Tips:
  • Check if Ollama is running in your computer.
  • Make sure the URL is for a single video on YouTube (nothing else).
  • Double check if your YouTube video has an actual transcript! (some videos may not have a transcript, such as classical music videos).
  • If your internet connection is slow, find a solution if possible to speed it up.


How It Works:
  1. When you submit a YouTube URL, we extract the video transcript using code.
  2. Next, we process the transcript with your local LLM, using a carefully crafted prompt to generate a response.
  3. Once the LLM finishes generating a response (it may take a while), we refine the output for clarity and readability..
  4. Finally, we deliver the polished response directly to you!
